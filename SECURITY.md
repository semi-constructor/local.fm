# Security Policy

## Supported Versions

The following versions of local.fm are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it directly to the maintainers. Do not create a public issue.

To report a vulnerability, please send an email to [toowake@icloud.com](mailto:toowake@icloud.com).

We will acknowledge receipt of your report and provide a timeline for addressing the issue. Once the issue has been resolved, we will work with you to coordinate a public disclosure.

Thank you for helping keep local.fm secure!
